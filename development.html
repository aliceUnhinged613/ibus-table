Development

Ibus-table is open source and licensed under the GPL version 2.1.

Browse git repository

You can browse the source code at https://github.com/{{ site.github_username }}/ibus-table.

Anonymous git clone

git clone git://github.com/{{ site.github_username }}/ibus-table.git

Authorized git clone

git clone git@github.com:{{ site.github_username }}/ibus-table.git

Building and installing using the source from git checkout

cd ibus-table
./autogen.sh --prefix=/usr
make
make install

Release tarballs

Release tarballs are available at https://github.com/{{ site.github_username }}/ibus-table/releases.

Building and installing using the source from a release tarball

tar xvf ibus-table-1.9.14.tar.gz
cd ibus-table-1.9.14
./configure --prefix=/usr
make
make install

Required packages

To build ibus-table, you will need at least: autoconf, automake, python >= 3.3, ibus >= 1.5.3
